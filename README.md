# SVG Optimizer

Customized SVGO-Node.js tool for optimizing SVG files.

Read more: [https://github.com/svg/svgo](https://github.com/svg/svgo)

## Usage

1. Paste your SVG into the `src` folder
2. Run optimization
    ```bash
    npm run optimize
    ```
3. Get your optimized SVG file in `dist` folder
